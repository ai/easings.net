Easing function especifica a velocidade do progresso da animação para tornar mais realista.

O objeto real não começa seu movimento instantaneamente e numa taxa constante. Quando abrimos uma gaveta, nós primeiro aceleramos e, depois, desaceleramos. Quando algo cai, isso primeiro cai rapidamente e, depois, quica de volta depois de chegar ao chão.

Essa pagina permitirá que você escolha a função easing desejada.

JavaScript

jQuery com Plugin jQuery Easing é a forma mais fácil de descrever a animação com easing. Você precisa apenas definir o nome da easing para o método .animate como terceiro argumento or chave easing.

div.animate({ top: '-=100px' }, 600, 'nome do easing', function () { … })

SCSS

Sass/SCSS te ajuda a descrever a animação. Compass remove prefixos antes das propriedades transition e animation e o plugin Compass Ceaser permite definir o easing pelo nome (sem curva Bezier).

div {
  @include transition(all 600ms ceaser($nome do easing); }

CSS

As propriedades CSS transition e animation te permite definir a função easing. Infelizmente, não dá suporte a todos os easings e você tem de definir a função pela curva Bezier.

Selecione a easing para mostar sua descrição na curva Bezier.

div {
  -webkit-transition: all 600ms curva Bezier da easing;
  -moz-transition:    all 600ms curva Bezier da easing;
  -ms-transition:     all 600ms curva Bezier da easing;
  -o-transition:      all 600ms curva Bezier da easing;
  transition:         all 600ms curva Bezier da easing; }

  @include transition(all 600ms ceaser($easeOutCirc); }

CSS

As propriedades CSS transition e animation te permite definir a função easing.

div {
  -webkit-transition: all 600ms cubic-bezier(0.075, 0.82, 0.165, 1);
  -moz-transition:    all 600ms cubic-bezier(0.075, 0.82, 0.165, 1);
  -ms-transition:     all 600ms cubic-bezier(0.075, 0.82, 0.165, 1);
  -o-transition:      all 600ms cubic-bezier(0.075, 0.82, 0.165, 1);
  transition:         all 600ms cubic-bezier(0.075, 0.82, 0.165, 1); }
