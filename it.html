La funzione Easing specifica la velocità delle accelerazioni di un'animazione, rendendola più realistica.

Un oggetto nella realtà non inizia istantaneamente il suo movimento e ad una frequenza costante. Quando apriamo il cassetto, prima gli imprimiamo un'accelerazione e poi lo facciamo rallentare. Quando qualcosa cade, per prima cosa scende sempre più veloce, fino a che non rimbalza indietro quando colpisce il pavimento.

Questa pagina ti aiuta ogni volta a scegliere la funzione easing desiderata.

CSS

Le proprietà CSS transition and animation ti permettono di impostare l'easing. Sfortunatamente, il loro supporto non è per tutte le easing, per questo devi impostare funzioni tramite la curva di Bezier.

Seleziona l'easing per mostrare la descrizione in curva di Bezier.

div {
  -webkit-transition: all 600ms curva di Bezier easing;
  transition:         all 600ms curva di Bezier easing; }

SCSS

Sass/SCSS ti aiuta a impostare le animazioni. Compass rimuove i prefissi prima delle proprietà transition e animation, il plugin Compass Ceaser ti permette di impostare l'easing con il suo nome (senza curve di Bezier).

div {
  @include transition(all 600ms ceaser($nome easing)); }

JavaScript

jQuery con il plugin jQuery Easing è il modo più semplice per impostare le animazioni con easing. Hai solo bisogno di inserire il nome dell'easing nella funzione .animate come terzo parametro o come chiave easing.

div.animate({ top: '-=100px' }, 600, 'nome easing', function () { … })

SCSS

Sfortunatamente, CSS non supporta questo easing. Puoi però utilizzarlo dentro JavaScript oppure con le Animazioni CSS speciali @keyframes.

JavaScript

jQuery con il plugin jQuery Easing è il modo più semplice per impostare le animazioni con easing. Hai solo bisogno di inserire il nome dell'easing nella funzione .animate come terzo parametro o come chiave easing.

div.animate({ top: '-=100px' }, 600, 'easeInElastic', function () { … })
